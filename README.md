# OriginStory / NextForm

Opinionated portfolio + community platform built with Next.js App Router, Auth.js, and Prisma.

## Quick start

```bash
npm install
npm run dev
```

## Environment

```
DATABASE_URL=
NEXTAUTH_SECRET=
GITHUB_CLIENT_ID=
GITHUB_CLIENT_SECRET=
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
```
